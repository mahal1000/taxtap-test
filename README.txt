TaxTap finished portable build

Open TaxTap.html in a browser.

Main included features:
- Home / Add / Play / Timeline / Settings
- calculator-style input
- top 5 usuals and repeat payments
- period-aware tax-year and quarter reporting
- Prepare figures review for quarterly and year-end views
- editable tax-year rule engine in Settings
- save my data / load my data backup buttons in Settings

This is a self-contained local browser build.
